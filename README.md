## ğŸ”® About The Project

This project is a solution to the **Stats Preview Card Component** challenge from **Frontend Mentor**.

The goal of this challenge was to build a **responsive card component** based on a provided design, focusing on layout, typography, and responsiveness.

What this project demonstrates:
- Clean and semantic HTML structure
- Responsive design using modern CSS
- Mobile-first workflow
- Attention to visual details and spacing
- Pixel-perfect implementation from a design brief
