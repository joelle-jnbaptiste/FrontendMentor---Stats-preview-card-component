## ğŸ¯ About The Project

This project is a solution to the **Stats Preview Card Component** challenge from **Frontend Mentor**.

The goal of this challenge was to build a clean and responsive UI component based on a provided design, with a strong focus on:

- Semantic and accessible HTML structure
- Modern CSS layout techniques
- Responsive behavior across screen sizes
- Attention to spacing, typography, and visual hierarchy
- Pixel-perfect implementation from a design brief

## âš”ï¸ Getting Started

This project is a static frontend component and does not require a backend.

### Prerequisites

- A modern web browser
- (Optional) A local development server

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/FrontendMentor---Stats-preview-card-component.git

2. Navigate to the project folder:

       cd FrontendMentor---Stats-preview-card-component

3. Open `index.html` directly in your browser  
   or serve it using a local development server.
